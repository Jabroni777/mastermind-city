# Mastermind City

Mastermind City is a multi-page concept website for a federated, opt-in system of
parallel communities focused on cooperative living, training centers, and practical
institution design.

## Project Structure

- `index.html` - Vision, strategic framing, and interactive federation simulator
- `model.html` - Governance + finance architecture and incentive mechanics
- `charter.html` - Mastermind Pod Charter, roles, rhythms, and pilot framework
- `technology.html` - Autonomous agent concepts and navigator bot demo
- `participate.html` - Participation pathways and application experience
- `styles.css` - Shared visual system and responsive design
- `script.js` - Shared interactions (menu, reveal, simulator, tabs, accordion, forms, bot)

## Run Locally

Open `index.html` directly, or run a lightweight local server:

```bash
python3 -m http.server 8080
```

Then visit <http://localhost:8080>.
