Internal Governance Laboratory

A simulator for finding viable integrity architectures.

This page is now oriented toward experimental analysis, not roleplay. It treats individuals as proxies for future nodes and lets you vary governance parameters, run large Monte Carlo batches, and compare which configurations most reliably preserve cooperation.

Design intent: identify which combinations of sociocratic structure, accusation thresholds, sanction severity, participation quality, and federation feedback produce the highest success rates without legitimacy collapse.

What this lab is for

Parameter sweeps

Adjust variables that plausibly matter, then run batches to see which configurations survive sabotage and avoid autoimmune overreaction.

Comparative models

Run the same configuration against baseline majority, sociocratic circles, and hybrid integrity governance.

Visual + statistical output

Use the single-run viewer to inspect dynamics, then use batch analysis to compare outcome distributions and success rates.

Experiment Controls

Configure the system you want to test.

Presets are just starting points. The important part is the parameter surface: population size, adversary count, accusation threshold, sabotage intensity, sanction costs, reward for correct enforcement, participation noise, and minimum support for federation review.
